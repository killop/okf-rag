# okr-rag-workspace

User workspace for OKR-RAG.

This directory is source truth. OKR Markdown files that should be indexed live under:

```text
okr-rag-workspace/okrs/
```

The demo OKR file is:

```text
okr-rag-workspace/okrs/domain-driven-memory-okr.md
```

Do not put generated indexes, caches, or MCP runtime state here. Those belong under `.okr-rag/`.
